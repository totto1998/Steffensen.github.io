Aplicativo

Instalación

Para utilizar la aplicación, sigue los siguientes pasos:

  1. Instala Python en tu sistema si aún no lo has hecho.
  2. Instala las dependencias necesarias ejecutando el siguiente comando en tu terminal:
    3. pip install flask
  3. Descarga el código fuente de la aplicación.
  4. Accede a la carpeta donde se encuentra el código fuente desde la terminal.
  5. Ejecuta el siguiente comando para iniciar la aplicación:
    6. python app.py
  6. La aplicación estará disponible en la dirección http://localhost:5000.

Uso de la aplicación

Una vez que la aplicación esté en funcionamiento, sigue estos pasos para utilizarla:

  1. Abre tu navegador web y accede a la dirección http://localhost:5000.
  2. Ingresa la fórmula de la ecuación que deseas resolver.
  3. Ingresa el valor inicial, la tolerancia y el número máximo de iteraciones.
  4. Haz clic en el botón "Calcular".
  5. Los resultados se mostrarán en la misma página, incluyendo el número de iteraciones, la aproximación obtenida y el valor de la función en la aproximación.
  6. Se generará una gráfica que muestra la evolución del método.
